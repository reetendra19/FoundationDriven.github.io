Who are we?
We're a newly formed group of minecraft enthusiasts that want to make it easier to develop minecraft plugins.
What platforms do you develop for?
We develop for Sponge, A relatively new API that's incredibly lightweight and flexible.
How can I help?
All of our code is hosted on github, click here to go to our organisation page.
I found a bug in one of your projects!
We have our own issue tracker so that we can keep track of all the bugs, simply find the project on the projects page and hit the issues button, this will take you to the issue tracker for that project. For a tutorial on how to use our issue tracker click here.
I have an idea for a feature / a new project!
Simply create an issue (see above) with details on your idea.
I don't code, Is there any other way I can help out?
Of course! You can find a full list of other ways you can contribute here.
